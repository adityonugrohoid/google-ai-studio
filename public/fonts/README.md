# Font Files Directory

## Neue Haas Grotesk

To use the premium Neue Haas Grotesk font, place your font files here:

- `NeueHaasGrotesk-Roman.woff2` (Regular - 400)
- `NeueHaasGrotesk-Roman.woff` (Regular - 400, fallback)
- `NeueHaasGrotesk-Medium.woff2` (Medium - 500)
- `NeueHaasGrotesk-Medium.woff` (Medium - 500, fallback)
- `NeueHaasGrotesk-Bold.woff2` (Bold - 700)
- `NeueHaasGrotesk-Bold.woff` (Bold - 700, fallback)

## Where to Get the Font

1. **Purchase from Linotype**: https://www.linotype.com/font/neue-haas-grotesk
2. **Adobe Fonts**: If you have an Adobe subscription
3. **Alternative**: The app will automatically fallback to Helvetica Neue (very similar) if font files are not found

## Testing Without Font Files

The app is configured to use Helvetica Neue as a fallback, which is very similar to Neue Haas Grotesk and will give you a good preview of the design.
